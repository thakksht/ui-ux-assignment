Programming Competitions

  • CodeMaster Challenge

    Description: A 6-hour coding marathon testing algorithmic skills and problem-solving abilities.

    Team Size: Individual

    Prize: ₹15,000 (1st), ₹10,000 (2nd), ₹5,000 (3rd)

    Languages Allowed: C++, Java, Python, JavaScript

  • Hackathon Supreme

    Description: 24-hour hackathon to develop innovative solutions for real-world problems.

    Team Size: 2-4 members

    Prize: ₹20,000 (1st), ₹12,000 (2nd), ₹8,000 (3rd)

    Theme: Sustainable Technology Solutions

  • Web Development Contest

    Description: Create a fully functional website within 8 hours.

    Team Size: 1-3 members

    Prize: ₹8,000 (1st), ₹5,000 (2nd), ₹3,000 (3rd)

    Requirements: Responsive design, modern UI/UX

Robotics & Hardware

  1. Robo Race

    Description: Navigate autonomous robots through challenging obstacle courses.

    Team Size: 2-5 members

    Prize: ₹12,000 (1st), ₹8,000 (2nd), ₹4,000 (3rd)

    Robot Specs: Maximum weight 5kg, autonomous navigation required

  2. Line Following Bot

    Description: Build robots that can follow predetermined paths with highest accuracy.

    Team Size: 2-4 members

    Prize: ₹6,000 (1st), ₹4,000 (2nd), ₹2,000 (3rd)

    Track: Complex path with multiple turns and obstacles

  3. Arduino Innovation

    Description: Create innovative IoT solutions using Arduino and sensors.

    Team Size: 1-3 members

    Prize: ₹7,000 (1st), ₹4,500 (2nd), ₹2,500 (3rd)

    Focus: Smart home, environmental monitoring, or automation
